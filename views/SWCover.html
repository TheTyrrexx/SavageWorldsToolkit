Savage Worlds Toolkit!
For all your gaming needs.

Welcome to the Savage Worlds Toolkit.
A collection of different tools to help Savage Worlds Game Masters and players get the maximum
enjoyment out of role playing. The Toolkit provides essential functions to make tabletop game
sessions run as smooth as possible. Accessible via laptop, tablet, or cell phone means that you can
always be ready for a game at a moments notice. No more needing to always remember pencils, dice, or
character sheets. Try integrating a little bit of technology with your tabletop gaming,
and sign up for an account, completely FREE!

Current Features:
  • Dice Roller
  • Character Sheets and Manager
  • Item Generator
  • Full Information Table References
  • Cloud Save Functionality
Planned Features:
  • Dungeon Generator
  • Extra/NPC Generator
  • Map Tools
  • Combat Grid and Initiative Tracker
